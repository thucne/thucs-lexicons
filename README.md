## Getting Started

Install dependencies

```bash
npm i
# or
yarn i
# or
pnpm i
# or
bun install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
